SSRF-内网Redis getshell

Redis是常见的内网服务，因其默认配置未授权访问或使用弱口令认证，可以导致getshell。Redis常见的getshell方式有以下几种:

  1. 绝对路径写Webshell
  2. 写ssh公钥
  3. 写crontab计划任务反弹shell
  4. 主从复制(4.0 < Redis < 5.0.5)
